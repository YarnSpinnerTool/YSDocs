## New Language Features in Yarn Spinner 3

* [`once` statements, allowing you to only run content a single time](../../new-in-v3/language-features/once.md)
* [`detour` statements, for briefly jumping to another node, then returning](../../new-in-v3/language-features/detour.md)
* [line groups, allowing you to create a collection of lines that Yarn Spinner will choose from for you](../../new-in-v3/language-features/line-groups.md)
* [node groups, for creating a collection of nodes that Yarn Spinner will choose from under certain conditions](../../new-in-v3/language-features/node-groups.md)
* [an `enum` type, for defining variables with a pre-defined constrained set of possibilities](../../new-in-v3/language-features/enums.md)
* [smart variables that determine their value at run-time](../../new-in-v3/language-features/smart-variables.md)
* [shadow lines for reusing lines](../../new-in-v3/language-features/shadow-lines.md)
