# NodeDestination.NodeName

Property in [NodeDestination](/docs/api/csharp/yarn.compiler.basicblock.nodedestination.md)

## Summary


The name of the node that this destination refers to.


```csharp
public string NodeName { get; set; }
```

## Remarks
