# LetterTypewriter.CharactersPerSecond

Property in [LetterTypewriter](/docs/api/csharp/yarn.unity.lettertypewriter.md)

## Summary


The number of characters per second to deliver.


```csharp
public float CharactersPerSecond { get; set; }
```

## Remarks

If this value is zero, all characters are delivered at
once, subject to any delays added by the markup handlers in  <a href="yarn.unity.lettertypewriter.actionmarkuphandlers.md">ActionMarkupHandlers</a> .
